用户权限

用户权限

打开创建好的版本库目录，修改 conf 目录下的 svnserve.conf 文件。

[general]
anon-access = none
auth-access = write
password-db = passwd
authz-db = authz

然后在同目录下的 passwd 文件中添加用户名和密码。

[users]
kuriv = 123456

最后在同目录下的 authz 文件中配置用户权限。

[/]
kuriv = rw
